# Style Index

This file is kept for compatibility. For XiaoShui Pretty PPT, prefer reading `ppt-template-catalog.md` first because it groups templates by presentation scenario.

## Pastel Blockfolio

- Chinese name: `粉彩拼贴志`
- Slug: `pastel-blockfolio`
- Module: Creator, personal brand, portfolio
- Best for: self-media tutorials, workflow case studies, process explanations, visual recaps, portfolio-like editorial decks.
- Visual language: off-white paper, thick black borders, pastel pink/mint/yellow blocks, sticky-note accents, compact section slides.
- Template: `assets/templates/pastel-blockfolio/index.html`
- Reference: `references/pastel-blockfolio.md`
