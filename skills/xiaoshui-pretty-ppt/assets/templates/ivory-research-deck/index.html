Page 03
Abstract

HTML is no longer only a web page.

核心观点：当内容需要持续更新、带交互、能被部署、能被朋友直接打开时，HTML 比传统 PPT 更适合作为“可展示的内容容器”。

用 Markdown 或文档生成 HTML，可以把“写作、排版、演示、发布”串成一条更轻的工作流。

内容可以从 Markdown、飞书文档、网页资料、图片和视频继续扩展。

PPT to HTML
Page 04

Why reports move from PPT to HTML.

传统 PPT 适合线下逐页讲，但当内容需要被二次传播、持续更新、插入视频、保留交互、在线访问时，HTML 会更像一个“可演示的网页文档”。

PPT

适合固定页数、现场汇报、单向讲述。

MD

适合先写内容、整理结构、快速修改。

HTML

适合加入交互、动画、目录跳转和媒体内容。

URL

适合部署分享，让对方无需下载文件即可查看。

What this HTML template can carry.

它可以承载偏学术的论点、职场汇报的数据、课程内容的章节、产品说明的流程，也可以把图片和视频放到合适的位置。

Media aware

图片、图表、视频可以进入不同页面模块。

Stable style

每次生成都沿用同一套学术纸面风格。
